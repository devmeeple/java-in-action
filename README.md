# Java in Action

## TDD Bowling Kata

- [간단한 볼링 게임 예제](http://butunclebob.com/ArticleS.UncleBob.TheBowlingGameKata)를 통해 TDD가 어떻게 이뤄지는지 알 수 있는 자료였다.
- 볼링 룰에 미숙하여 구현 동작을 완벽하게 이해하기는 실패했다. 
- 우선 게임의 룰과 요구사항을 숙지하고 반복하면 좋을 것 같다.
